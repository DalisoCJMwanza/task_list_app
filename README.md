# Task List Application

## Project Description
This is a simple Python Task List application developed to demonstrate collaborative development using Git and GitHub.

## Features Implemented
- Add tasks
- Due dates for tasks
- Reoccuring task implementation
- Priority levels (High, Medium, Low)
- Search tasks
- Mark tasks as completed
- Deleting tasks

## Technologies Used
- Python
- Git
- GitHub

## Git Workflow Used
Each feature was implemented in a separate branch:
- Due dates
- Delete task
- feature-due-dates
- feature-search
- feature-priority
- feature-status

These branches were later merged into the main branch.

## How to Run the Application

Run the following command:

python app.py

Then follow the menu options to manage tasks.
